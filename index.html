How Web Works Exercise

Part One

  • What is HTTP?
    • Hypertext Transfer Protocol (standard web) (How browsers and servers communicate)

  • What is a URL?
    • URLs denote a protocol, the hostname, a port, a resource accessed and a query

  • What is DNS?
    • DNS is a database that associates hostnames with ip addresses

  • What is a query string?
    • Query strings are data that is queried to a webserver. Format varies from each website

  • What are two HTTP verbs and how are they different?
    • GET makes a request that has no side effiects
    • POST makes a request that has side effiects via arguments sent

  • What is an HTTP request?
    • A GET request with a target hostname and URL

  • What is an HTTP response?
    • The data you requested, metadata, and the status code

  • What is an HTTP header? Give a couple examples of request and response headers you have seen.
    • The metadata sent along with a webpage
    • Hostnames, Date, Language, Cookies, and more

  • What are the processes that happen when you type “http://somesite.com/some/page.html” into a browser?
